DOELT: Tracking For Better Tomorrow

Software Engineering, JavaScript, React

OVERVIEW

A project created for the given challenges in the Hawaii Annual Code Challenge 2022. The challenge was to create an application in which it automatically extracts legislations and displays the status to the user. The application placed second in the code challenge.
  • For the Github Repository
  • For the Project Group Page

TECHNICAL SHEET

The website was created with the following details

  • Due to the time constraints used the UH Manoa's ICS template
  • Front end was built using JavaScript React and React Bootstrap as its framework
  • The Backend was built using Meteor and MongoDB to store data

A video discussing about the application.

TASKS

Majority of my responsibilities was front end designs. I began by creating rough designs using Figma and then created the design so it was appealing both from Desktop and Mobile view.

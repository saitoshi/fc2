Open Innovation: 0147

MERN Stack, TypeScript
November 2022 to ONGOING

OVERVIEW

This application was created as a project for Kumamoto City under Local Venture Room INC. The application stores "Open Innovation" based project.
Users can create accounts and register their innovation ideas or submit their solutions.

TECHNICAL SHEET

The website was MERN Stack with the following details
  • The design layout was created using Figma.
  • The frontend used TypeScript React with Bootstrap React as its framework.
  • The backend used TypeScript, Express, & Node.JS
  • The database was stored using MongoDB.
  • The PDF files that users would upload or submit was stored to AWS S3 Bucket.
  • The code was written using Visual Studio Code.

TASKS

This was my first paid project using MERN stack development. At this point, my past project were meteor-based. Therefore, there was some hurdles that I faced. Most of my responsibilities at first was backend based. However, as it progressed my responsibilities also covered the front end side too.
  • Created the user-related APIs with the aid from MERN stack development manual
  • Connected the frontend and backend functions for account registeration, user login, other account related activities, and innovation submissions.
  • Connected the application to an AWS S3 Bucket to store PDF files
  • With the assistance of nodemailer, made it possible so users and management can receive emails related to innovation activities

This is an application created for Local Venture Room Inc. This application stores different events similar to Hacks from Hackathons registered by different companies in the Kumamoto Prefecture Area. A user will create an account or log in and then register for an event that interests them. Depending on the event, winners will be selected for a prize given by that specific company.

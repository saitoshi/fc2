FAN FUND

Software Engineering, TypeScript, React
August 2022 to ONGOING
One of the goals at Local Venture Room was to develop an app in which Japanese companies or projects can register their goals to receive crowd funding. However, instead of normal money, people will send in tokens that complies with the different laws in Japan.
BACKGROUND
The logo of the app.
RELATED SYSTEM
There are similar apps that have the similar goals with Fan Fund. Despite the similarities, Fan Token still stands out with its differences.
  • FINANCIE
    While FINANCIE is an app with the similar concept (users can send in tokens to projects or goals), the main difference is that FINANCIE is limited to sports groups. In Fan Fund, we aim to widen the inclusivity of different genres.
PROJECT DEVELOPMENT
The project development took usage of GitHub's project board by breaking it into phases. Furthermore, each tasks (or issues) were labeled with the following:
  • FRONTEND
    Issues that are frontend related creation or updates such as fixing the size of certain fonts or adding buttons.
  • BACKEND
    Issues that are backend related tasks such as API or server related.
  • BOTH
    Issues that dealt with both frontend & backend related updates or fixes.

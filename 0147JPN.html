Open Innovation: 0147

MERN Stack, TypeScript
November 2022 to ONGOING

OVERVIEW

熊本市 のために、オープンイノベーションをまとめたサイトをLocal Venture Room INC.が作成した
基本的にはユーザーはメールアドレスで会員登録をし、開催されているイノベーションへ提案を提出する流れがベースとなっております。

TECHNICAL SHEET

システムはMERNベースで作成されました。以下が細かい内容です。
  • デザインレイアウトはFIGMAで作成されました
  • フロントエンドはTypeScript ReactとReact Bootstrapをフレームワークとして作成されました。
  • バックエンドはTypeScript, Express, Node JSで作成されました。
  • データーベースはMongoDBで登録されています。
  • ユーザーが提出するPDF類は全て、AWSのS3 Bucketに保存されております。
  • コード作業は全てVisual Studio Code とGitHubをチームとして使用しました

TASKS

大学の研究、インターン、仕事は大体Meteorベース又はWordPressを使用していたサイトマネジメントだったために、今回が初Mernのプロジェクトでした。
基本的にはバックエンドを担当をしていましたが、フロントとバックのリンクの作成やテストなども行いました。今回のプロジェクトは初めての経験も多かったために次のプロジェクトへの 反省点は結構ありますが、今度どのようにプロジェクトマネージメントへ活かしたいと思います。
以下が0147で担当した作業になります。
  • YouTubeやネットにあるMERNマニュアルを参考に、ユーザーやイノベーション関係のAPI作成
  • ログインページと会員登録ページをバックエンドと接続するように設定を変更する
  • MongoDBでデーターベース用のSchemaとCollectionの作成
  • ユーザーのパスワード変更やパスワードを忘れた場合のフロントとバックエンド関係の作成
  • AWSとのリンクを作成、それにより必要なPDFファイルがバケットに保存される
  • NodeMailerを使用し、ユーザー関係でイベントが起きると、運営側とユーザー側が通知するように設定
